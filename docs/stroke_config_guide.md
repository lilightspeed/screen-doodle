## velocity — 速度敏感宽度

控制画笔宽度如何随绘画速度变化的速度映射行为。仅对 **Pen / Pen2 / Pen3** 生效，荧光笔和橡皮擦不受影响。

### `sample_interval`

- **类型**: 整数
- **默认值**: `4`
- **范围**: 1 ~ 16

鼠标事件采样间隔。每经过 N 个鼠标事件才重新计算一次笔迹宽度，中间的点复用上一次的计算值。

| 值 | 效果 |
|----|------|
| 1 | 每帧都计算，响应最快但最抖（不推荐） |
| 4 | 均衡，手感稳定同时带宽敏感 |
| 8 | 宽度变化更平缓，适合追求稳定手感 |
| 16 | 响应很迟钝，宽度几乎恒定 |

> **建议**: 从 4 开始调。觉得宽度变化太快 → 增大；觉得不够跟手 → 减小。

### `smoothing_alpha`

- **类型**: 浮点数
- **默认值**: `0.12`
- **范围**: 0.01 ~ 0.5

指数平滑系数。控制当前计算的新宽度值以多快速度替代旧值。**数值越小，宽度过渡越平滑但响应越慢**。

公式：`new_width = alpha × current + (1 - alpha) × previous_width`

| 值 | 效果 |
|----|------|
| 0.05 | 过渡非常柔滑，宽度变化有"慢半拍"的拖尾感 |
| 0.12 | 默认值，平滑与跟手的平衡点 |
| 0.25 | 过渡较快，适合追求即时反馈 |
| 0.50 | 过渡锐利，手速变化直接体现在笔迹上 |

> **建议**: 觉得笔迹粗细切换太突兀 → 调低；觉得宽度跟不上手速 → 调高。

### `thin_mult`

- **类型**: 浮点数
- **默认值**: `0.4`
- **范围**: 0.1 ~ 1.0

最快绘画速度下的宽度倍率。数值越小，快速画出的笔迹越细。

当前笔刷基础宽度 × `thin_mult` = 最快速度时的实际宽度。

> **建议**: 设为 0.25 可以获得更极端的粗细对比，设为 0.7 则快速画出的差异更温和。

### `thick_mult`

- **类型**: 浮点数
- **默认值**: `2.5`
- **范围**: 1.0 ~ 5.0

最慢绘画速度下的宽度倍率。数值越大，慢速画出的笔迹越粗。

当前笔刷基础宽度 × `thick_mult` = 最慢速度时的实际宽度。

> **建议**: 设为 3.0~4.0 可以大幅突出慢速时的厚重感。

### `ref_dist`

- **类型**: 浮点数
- **默认值**: `20.0`
- **范围**: 5.0 ~ 100.0

速度曲线的参考距离（像素）。距离 = 两个相邻鼠标事件之间的欧几里得距离。当距离等于 `ref_dist` 时，宽度倍率大约处于 `thin_mult` 和 `thick_mult` 的中间位置。

| 值 | 效果 |
|----|------|
| 10 | 曲线在较小的速度下就达到薄端，快速笔画的宽度差异变小 |
| 20 | 默认，均衡范围 |
| 40 | 要达到极薄需要更快速度，中速区域宽窄变化更丰富 |

> **建议**: 觉得画快画慢变化太大 → 增大 ref_dist。觉得变化不够明显 → 减小。

### `power_exponent`

- **类型**: 浮点数
- **默认值**: `0.7`
- **范围**: 0.1 ~ 1.0

速度曲线的幂指数，控制速度响应曲线的**形状**。

- **< 1.0**: 在低速区域分配更多敏感度（默认 0.7：低速段变化丰富，高速段趋于平稳）
- **= 1.0**: 线性映射，速度变化均匀体现在宽度上
- **> 1.0**: 在高速区域分配更多敏感度

> **建议**: 觉得慢速画的时候宽窄变化不够丰富 → 调低到 0.5。觉得低速段变化太敏感 → 调高到 0.9。

## rendering — 渲染品质

控制笔迹在屏幕上的渲染表现。

### `min_segment_width`

- **类型**: 浮点数
- **默认值**: `0.5`
- **范围**: 0.1 ~ 3.0

逐段绘制时的最小段宽度（像素）。防止因速度计算为零宽度导致的渲染断裂或锯齿。

| 值 | 效果 |
|----|------|
| 0.1 | 几乎不限制，极快画速下可能出现断点 |
| 0.5 | 默认，安全的最小值 |
| 1.5 | 快速笔划的最小宽度较厚，视觉上更稳定 |
| 3.0 | 限制了粗细对比的上限 |

> **建议**: 如果在极快画速下看到笔迹断断续续，适当增大此值。

### `preview_opacity`

- **类型**: 浮点数
- **默认值**: `0.7`
- **范围**: 0.1 ~ 1.0

绘画过程中的笔迹预览透明度。落笔后到松手之前看到的笔迹是预览状态。

| 值 | 效果 |
|----|------|
| 0.4 | 预览很淡，便于观察背景内容 |
| 0.7 | 默认，半透明预览 |
| 1.0 | 预览与最终笔迹完全一致 |

> **建议**: 仅在需要更清楚地看到正在画的内容时才调整。

### `highlighter_opacity_scale`

- **类型**: 浮点数
- **默认值**: `0.3`
- **范围**: 0.05 ~ 1.0

荧光笔模式的额外透明度倍率。最终透明度 = 笔刷不透明度 × `highlighter_opacity_scale`。

> **建议**: 设为 0.15 获得更通透的荧光效果，设为 0.5 获得更实在的覆盖效果。

### `highlighter_width_scale`

- **类型**: 浮点数
- **默认值**: `4.0`
- **范围**: 1.0 ~ 10.0

荧光笔模式的宽度倍率。荧光笔最终宽度 = 荧光笔基础宽度 × `highlighter_width_scale`。

> **建议**: 调小到 2.0 可以获得更精细的荧光标注效果，调大到 6.0 更醒目。

## 参数速查表

| 参数 | 默认值 | 范围 | 调大 → | 调小 → |
|------|--------|------|--------|--------|
| `sample_interval` | 4 | 1 ~ 16 | 更平缓 | 更跟手 |
| `smoothing_alpha` | 0.12 | 0.01 ~ 0.5 | 更锐利 | 更柔滑 |
| `thin_mult` | 0.4 | 0.1 ~ 1.0 | 快速画也更粗 | 粗细对比更大 |
| `thick_mult` | 2.5 | 1.0 ~ 5.0 | 慢速画更厚 | 粗细对比更小 |
| `ref_dist` | 20.0 | 5.0 ~ 100.0 | 需要更快才能变薄 | 慢速变化更丰富 |
| `power_exponent` | 0.7 | 0.1 ~ 1.0 | 高速段变化多 | 低速段变化多 |
| `min_segment_width` | 0.5 | 0.1 ~ 3.0 | 防止断裂 | 允许更极致的细 |
| `preview_opacity` | 0.7 | 0.1 ~ 1.0 | 预览更实 | 预览更透 |
| `highlighter_opacity_scale` | 0.3 | 0.05 ~ 1.0 | 荧光更实在 | 荧光更通透 |
| `highlighter_width_scale` | 4.0 | 1.0 ~ 10.0 | 荧光更宽 | 荧光更精细 |

## 调优示例

### 🎯 追求极致的粗细对比

```json
{
  "velocity": {
    "thin_mult": 0.2,
    "thick_mult": 4.0,
    "ref_dist": 15.0,
    "power_exponent": 0.5
  }
}
```

### 🎯 平滑温和的变化

```json
{
  "velocity": {
    "sample_interval": 6,
    "smoothing_alpha": 0.08,
    "thin_mult": 0.5,
    "thick_mult": 2.0,
    "ref_dist": 30.0
  }
}
```

### 🎯 快速跟手、变化明显

```json
{
  "velocity": {
    "sample_interval": 3,
    "smoothing_alpha": 0.25,
    "thin_mult": 0.3,
    "thick_mult": 3.0,
    "ref_dist": 10.0
  }
}
```

### 🎯 荧光笔加强（更透、更宽）

```json
{
  "rendering": {
    "highlighter_opacity_scale": 0.15,
    "highlighter_width_scale": 6.0
  }
}
```

## 故障排除

| 现象 | 原因 | 方案 |
|------|------|------|
| 笔迹断裂、有空白点 | `min_segment_width` 过小 | 增大到 1.0 ~ 1.5 |
| 笔迹宽度变化太突兀 | `smoothing_alpha` 过大 或 `sample_interval` 过小 | 降低 alpha 到 0.08 或增大采样间隔到 6 |
| 宽度变化太慢、不跟手 | `smoothing_alpha` 过小 或 `sample_interval` 过大 | 提高 alpha 到 0.2 或减小采样间隔到 2 |
| 快画慢画宽度差不多 | `thin_mult`/`thick_mult` 差异不够大，或 `ref_dist` 过大 | 拉开 thin/thick 差距，或减小 ref_dist |
| 配置文件报错、应用打不开 | JSON 格式错误 | 删除 `stroke_profile.json` 重新生成，或检查逗号/引号是否完整 |
